Per l'esercizio ho preso spunto da "temperature.2.xml" in "lweb-part10-XML2.pars2".

Nell'esercizio si vuole sperimentare l'uso di XML con DOM. Ho usato dtd.

Nel compito ho voluto simulare una "ipotetica" pagina di un sito di scommesse.
All' apertura di "sport.php" vengono elencati una serie di eventi(calcio, basket e tennis).
Attraverso una selezione è possibile filtrare gli eventi per sport(pulsante Invio). Poi è possibile
tornare alla lista originale, aggiungere alla "schedina" le scommesse selezionate, eliminare le scommesse aggiunte e
scommettere(viene visualizzato soltanto un messaggio in blu).
